# Rethinking Real-Time Collaboration: A Local-First Approach

This is a research poster made for the UCSC undergraduate course CSE 185S (Technical Writing and Communication in Computer Science) exploring local-first software as an alternative model for real-time collaborative tools. While cloud apps are widely treated as the default model for software currently, they restrict users' control and ownership of their data and create user dependence on vendors. Local-first software instead keeps the primary copy of data on the user's device, enabling offline capability, instant responsiveness, and greater user ownership, among other benefits, without sacrificing collaboration.
